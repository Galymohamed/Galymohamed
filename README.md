# 💫 About Me:
🔭 I'm currently working on GalyTEK (galytek.com) — a full-stack web platform built with Java Spring Boot & Thymeleaf, where I document and publish my embedded systems projects, firmware experiments, and technical writeups.<br>🤝 I'm looking for help with growing the GalyTEK content library — if you work on embedded systems and want to contribute technical articles or projects, let's connect.<br>💬 Ask me about STM32 bare-metal & FreeRTOS development, low-level peripheral drivers (SPI, I²C, CAN, UART, DMA), AUTOSAR-compliant software development, ECU diagnostics (UDS/ISO 14229), PCB design in KiCad, and Embedded Linux on BeagleBone.<br>🌱 I'm currently learning advanced Embedded Linux, device driver development, and exploring safety-critical systems (IEC 61508 / ISO 26262).<br>⚡ Fun fact: I built GalyTEK from scratch using Java Spring Boot — an embedded engineer who also codes full-stack web applications!
